01 — User Journey
End-to-End User Flow

The complete journey a user takes from landing on the homepage to sharing their trip with friends.

1
Discover
Land on Home page. Browse popular destinations. Read features overview.
2
Plan
Search destination with autocomplete. Set travelers, days, budget, category.
3
Explore
View attractions, hotels, and a full day-by-day itinerary with images.
4
Save
Create an account, wishlist trips, revisit history from the dashboard.
5
Share
Generate a public share link. Send via WhatsApp, Facebook, Instagram, TikTok & more.
Note: Steps 1–3 are available to guests (no login required). Steps 4–5 require an authenticated account. Shared trip links (/shared/{token}) are publicly accessible — no login needed to view.

03 — Data Flow
Request & Response Architecture

How a trip generation request flows from the browser through the backend and back.

🖥️ Browser
📍
DestinationSearch
Debounced GET /api/places/autocomplete?input=…
📋
TripPlannerForm
POST /planner/generate with validated form data + lat/lon
❤️
Wishlist / Share / Delete
Axios POST/DELETE with CSRF token header
→
⚙️ TripController
🔍
autocomplete()
Delegates to GeoapifyService → returns JSON suggestions
🚀
generate()
Validates input → calls TravelApiService → saves Trip → renders Inertia page
🔗
generateShareLink()
Creates unique share_token (Str::random 48) → returns public URL
🌐
showShared()
Finds trip by token → renders read-only TripResults (no auth)
→
🔧 Services
🌍
GeoapifyService
autocomplete, geocode, getNearbyPlaces, getNearbyHotels
✈️
TravelApiService
generateTripPlan → real attractions + hotels + itinerary + overview
→
🌐 External APIs
📡
Geoapify Autocomplete
api.geoapify.com/v1/geocode/autocomplete
🗺️
Geoapify Places
api.geoapify.com/v2/places — nearby attractions & hotels
🖼️
Picsum Photos
Seed-based placeholder images for destinations, hotels, itinerary
→
🗄️ SQLite DB
👤
users
id, name, email, password, timestamps
✈️
trips
destination, travelers, days, category, budget, attractions JSON, hotels JSON, itinerary JSON, wishlisted, share_token
🏨
bookings
trip_id, type, name, price, date
04 — Share Feature
Trip Sharing System

How a trip gets shared and viewed by anyone, without requiring them to log in.

🔐
Owner clicks Share
Only the trip owner sees the Share button in TripResults hero or Dashboard card
→
⚙️
POST /trips/{id}/share
Controller checks ownership (403 if not owner). Generates 48-char random token if none exists
→
🔗
URL returned
Returns /shared/{token} — unique, permanent public URL
→
📤
Modal opens
Copy button + 6 share platform options displayed in a modal overlay
→
👁️
Anyone views
Recipient opens the link — sees full trip in read-only mode. No login required

Geoapify
Location APIs — autocomplete, geocoding, nearby places & hotels (real data)
🖼️
Picsum Photos
Seed-based placeholder images — consistent per destination/hotel/itinerary day
08 — API Integrations
External Services

The third-party APIs integrated into GlobePlanner and how they are used.

Location
Geoapify
Powers destination search autocomplete, city geocoding, and real nearby attractions & hotel discovery within a 10km radius.
GET /v1/geocode/autocomplete
GET /v1/geocode/search
GET /v2/places?categories=tourism…
GET /v2/places?categories=accommodation…
Images
Picsum Photos
Consistent placeholder images based on seed strings. Each destination, hotel, and itinerary day gets a unique, reproducible image.
GET picsum.photos/seed/{city}/1200/800
GET picsum.photos/seed/{name}hotel/800/600
GET picsum.photos/seed/{city}{day}/600/400
Search
Google Search
Used for external booking and ticket discovery. Clicking "Book Now" or "Search on Google" opens a new tab with a targeted Google search query.
GET google.com/search?q=book+{hotel}+hotel
GET google.com/search?q={attraction}+tickets
09 — Route Map
All Application Routes

Every URL in the application, its method, auth requirement, and what it does.

Method URI Controller Auth Description
GET / Closure → Home Public Landing page
GET /login AuthController@showLogin Guest only Show login form
POST /login AuthController@login Guest only Authenticate user
GET /register AuthController@showRegister Guest only Show register form
POST /register AuthController@register Guest only Create new account
POST /logout AuthController@logout Auth Destroy session
GET /planner TripController@index Public Show trip planner form
POST /planner/generate TripController@generate Public Generate trip plan (saves if logged in)
GET /api/places/autocomplete TripController@autocomplete Public Geoapify autocomplete proxy
GET /shared/{token} TripController@showShared Public Read-only shared trip view
GET /dashboard TripController@dashboard Auth Personal trips dashboard
GET /trips/{trip} TripController@show Auth + Owner View saved trip (replay results)
POST /trips/{trip}/wishlist TripController@toggleWishlist Auth + Owner Toggle wishlist boolean
POST /trips/{trip}/share TripController@generateShareLink Auth + Owner Generate/get share token & URL
DELETE /trips/{trip} TripController@destroy Auth + Owner Permanently delete trip
